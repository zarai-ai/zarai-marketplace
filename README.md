## Publication bar

An artifact enters this repository only when:

1. It is domain-general — no workstation-local paths, credentials, or private project internals.
2. It has been used in production on real work, with witnessed results.
3. It is export-appropriate: EchoAI Labs operates toward ITAR compliance; nothing defense-relevant, patent-pending-internal, or client-confidential is ever published here. Content is judged by what it actually contains, not by its metadata.
